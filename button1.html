LESSON 3

Let's begin using Javascript to update our button

Your Fancy Button from before has been reset using the .button2 class

The button now has an action, changing color when you click it.

The code inside the onclick attribute finds the element using id and then accesses the style settings.

document.getElementById('demo').setAttribute('style','background-color:rgb(210,50,35);')

 

 

This same code could be placed inside a function in your script file.

TIP: Always make sure your script file is linked at the top of your HTML document.
